Community rules + staff standards. Clear punishments, proper evidence, and clean logging so everything stays fair.

Community Rules

Applies to Discord + in-game
What this covers

This applies to all P.U.S.H spaces (Discord, events, game sessions, voice, and community-run chats).

Slurs / Hate Speech
  • Slurs or hateful language uses the ladder below (no “joke” exceptions).
  • Filter bypassing counts (spacing/symbols).
  • Targeting identity = not allowed.
1st
Warning
2nd
Mute / Timeout
3rd
Ban
Appealable: Yes (for ladder actions).
If it becomes personal threats / doxx / DDOS threats, it moves to zero-tolerance.
Zero-tolerance: threats / doxxing / DDOS threats
  • Real-life threats or safety threats
  • Doxxing or threatening to leak private info
  • DDOS threats (“boot you offline”, “hit your net”, etc.)
  • Coordinating harassment
Result: Permanent ban
Appealable: No
Staff must attach proof: screenshots/message links + timestamps + usernames/IDs
Harassment / bullying
  • No targeted abuse, stalking behavior, or dogpiling.
  • “In DMs” doesn’t protect it if it started through the community.

Staff Standards

PR / calm handling
How staff should act
  • Stay professional (no arguing like a member).
  • Enforce rules with evidence, not emotion.
  • Be consistent. Same rule = same action.
  • No leaking internal reports or staff chat.
When slurs happen
  • Stop it fast: “That’s not allowed here. Drop it now.”
  • Apply ladder: warn → mute → ban
  • Log the action + attach evidence
Info bar buttons
Reminder: If there’s no proof, don’t action. Ask for evidence first.

Logs & Evidence

Required for staff actions
What every log must include
  • Staff member name + role
  • User name + user ID (if possible)
  • Rule broken (which section)
  • Action taken (warn / mute / ban / perma)
  • Evidence: screenshots or message links + timestamps
  • Date/time + short context
No log = action can be reversed. Logging protects the community and protects staff too.

Appeals

Fair + consistent
Appealable
  • Warning / mute / ban ladder actions
  • Most non-zero-tolerance punishments
Not appealable
  • Perma bans for real-life threats, doxxing, or DDOS threats
Appeal should include
  • What happened (brief)
  • Why they think it was wrong / context
  • What they’ll do differently
